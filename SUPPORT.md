# Support

## Where to get help

| I need to... | Go here |
|---|---|
| Report a bug | [Open a bug report](https://github.com/Mobterest-Studio/dartform/issues/new?template=bug_report.yml) |
| Request a feature | [Open a feature request](https://github.com/Mobterest-Studio/dartform/issues/new?template=feature_request.yml) |
| Ask a question | [GitHub Discussions](https://github.com/Mobterest-Studio/dartform/discussions) |
| Get help with my subscription or billing | [support@dartform.dev](mailto:support@dartform.dev) |
| Report a security issue | [security@dartform.dev](mailto:security@dartform.dev) |
| Read documentation | [dartform.dev](https://dartform.dev/docs) |

## Before opening an issue

1. Make sure you're on the **[latest release](https://github.com/Mobterest-Studio/dartform/releases/latest)**
2. Search **[existing issues](https://github.com/Mobterest-Studio/dartform/issues)** — your question may already be answered
3. Check **[Discussions](https://github.com/Mobterest-Studio/dartform/discussions)** for community Q&A

## Response times

GitHub issues and discussions are monitored regularly. For urgent account or billing matters, email [support@dartform.dev](mailto:support@dartform.dev) directly for the fastest response.
